# art0-NYU

**Live:** https://art0-nyu.web.app

Generated experiences of artworks, notes made in 2013-2014.

---

## Tech Stack

- React 19
- TypeScript
- Vite
- Framer Motion
- Firebase Hosting

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `./local.sh` | Run locally |
| `./deploy.sh` | Deploy to Firebase |

## Features

- Progressive image loading with smooth fade-in
- Responsive images (4-tier srcset)
- 60fps+ GPU-accelerated animations
- Lazy loading with Intersection Observer
- Dynamic masonry layout

## Modes

- **Grid** - Living masonry grid with dynamic layouts
- **Reel** - Continuous scrolling experience
- **Deck** - Card-based navigation
- **Playground** - Interactive exploration
- **Overview** - Complete collection view
